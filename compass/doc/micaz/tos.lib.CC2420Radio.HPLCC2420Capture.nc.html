Commands - Details

enableCapture

result_t enableCapture(bool low_to_high)

Enable an edge based timer capture

Parameters:

low_to_high - TRUE if the edge capture should occur on a low to high transition, FALSE for high to low.

Returns: SUCCESS if the timer capture has been enabled

disable

result_t disable(void)

Diables a capture interrupt
Returns: SUCCESS if the interrupt has been disabled

Events - Details

captured

result_t captured(uint16_t val)

Fired when an edge interrupt occurs.

Parameters:

val - the raw value of the timer captured

Returns: SUCCESS to keep the interrupt enabled, FAIL to disable the interrupt
