Commands - Details

indexOf

result_t indexOf(uint16_t *pResult, uint16_t fromIndex, uint8_t *bitVec, uint16_t length)

Locates the index of the first '1' bit in a bit vector.

Parameters:

result - the location of the '1' bit

fromIndex - the index to start search for '1' bit

bitVec - the bit vector

length - the length of the bit vector in bits

Returns: SUCCESS if a '1' bit was found; FAIL otherwise.

countOnes

result_t countOnes(uint16_t *pResult, uint8_t *bitVec, uint16_t length)

Counts the number of '1' bits in a bit vector.

Parameters:

result - the number of '1' bits

bitVec - the bit vector

length - the length of the bit vector in bits

Returns: SUCCESS if the operation completed successfully; FAIL otherwise.
