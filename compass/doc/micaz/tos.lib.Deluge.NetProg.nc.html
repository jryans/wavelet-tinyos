programImgAndReboot

result_t programImgAndReboot(uint8_t imgNum)

Reboot into the image specified by imgNum. This assumes that an image has been downloaded into slot imgNum using Deluge.

Parameters:

imgNum - Number of image to boot in to.

Returns: FAIL if the reboot command fails to complete due to an invalid imgNum or incomplete image, does not return, otherwise.
