Interface: StatsArray

Computes various statistics on a data stream without storing the data itself.
Author: Ryan Stinnett

Components providing this interface:
apps.compass.StatsArrayM

Components requiring this interface:
apps.compass.StatsM

Commands

  • void newData(float newVal) Adds a new data element to the array.

  • float min(void) Returns the array's minimum value.

  • float max(void) Returns the array's maximum value.

  • float mean(void) Returns the mean of the array.

  • float median(void) Returns the estimated median of the array.

  • void clear(void) Removes all data from the array.
