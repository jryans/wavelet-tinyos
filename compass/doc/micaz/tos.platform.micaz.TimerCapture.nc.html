setSynchronous

void setSynchronous(bool synchronous)

Set whether the capture should occur synchronously or asynchronously. TinyOS default is synchronous captures.

Parameters:

synchronous - TRUE to synchronize the timer capture with the next timer clock instead of occurring asynchronously.
