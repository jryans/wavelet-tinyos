Interface: SharedMsgBuf

Simple component that allows sharing of message buffers through a parameterized interface.
Author: Jonathan Hui <jwhui@cs.berkeley.edu>

Commands

  • TOS_MsgPtr getMsgBuf (void) Get the pointer to a shared message buf.

  • void lock(void) Acquire lock on shared message buf.

  • void unlock(void) Release lock on shared message buf.

  • bool isLocked (void) Check if shared message buf is currently locked.

Events

  • void bufFree(void) Signal that the bus is free to allow other components waiting to begin using it right away.
