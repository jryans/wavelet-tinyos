Component: SensorControlM

This application takes data readings from various sensors, caches their values, and sends out the most recent set of data when requested. IMPORTANT: Temp and Light must start and stop with each request. If they are not stopped, the radio will not be able to receive.
Author: The Moters (Fall 2005 - Spring 2006)
