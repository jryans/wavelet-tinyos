Interface: Router

Allows other components access to various details of the unicast routing layer. Separating the routing layer makes it easy to test different routing strategies.

Commands - Details

getStatus

uint8_t getStatus(void)

Reports the current state of the routing layer. RO_INIT means the layer is starting up and determining the initial routing table. RO_READY means it has a usable routing table.
